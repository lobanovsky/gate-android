# RuStore Assets Checklist

## Обязательные материалы
- Store icon `512x512` в PNG
- Скриншот экрана входа
- Скриншот главного экрана с секциями `Паркинг` и `Двор`
- Скриншот состояния `Ждите...` на кнопке открытия
- Краткое описание
- Полное описание
- Контактный email разработчика
- Privacy Policy URL

## Рекомендации по скриншотам
- Используйте release-сборку, а не debug.
- Делайте скриншоты на чистом фоне без системных уведомлений.
- Покажите финальные тексты, цвета и иконки.
- Если на скриншотах есть реальные email или номера, убедитесь, что это допустимо для публикации.

## Файлы, которые должны быть готовы перед загрузкой
- Signed `.aab`
- Финальный текст из `RuStoreMetadataTemplate.md`
- Store icon `512x512`
- Набор скриншотов
