## App Title

## Short Description
Управление шлагбаумами и воротами с телефона.

## Full Description
Gate of Love — приложение для управления шлагбаумами и воротами.

Пользователь может:
- войти в приложение по email и паролю
- использовать сохранённую сессию и биометрический вход
- открывать въезд и выезд для зон `Двор` и `Паркинг`
- быстро звонить на резервные номера с главного экрана

Приложение работает с backend-инфраструктурой HouseKPR и повторяет ключевой сценарий iOS-версии.
